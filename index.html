App Architecture

Capsule Reviews is a Kotlin-based movie and tv show database application for Android. The app was developed with the MVVM architectural pattern.

Data Source

The data is provided by the The Movie DB Rest API (remote data source) using an API service build with OkHttp and Retrofit.

Repository Pattern

Capsule Reviews uses the repository pattern to manage the data comes from the remote and the local data sources.

Observer Pattern

View models provides latest data to the UI layer using the LiveData streams and the view state is updated asynchronously.

Features

Movie List

Displays a glimpse of search results for popular movie endpoints including popular, top-rated, upcoming, and trending movies.

Data is observed in the LiveData format and is displayed by a recycler view. Any user interaction over data is handled by the fragment and the view model.


Key Features


  • Displays top trending movie as a showcase

  • Inserts showcase movie into local Favorite database

  • Plays showcase movie trailer on YouTube

  • Displays multiple lists of movies from various endpoints

Paged Movie List

Displays an endless scrolling list of movies using the Paging Library for popular movie endpoints including popular, top-rated, upcoming, etc. along with movies by genre.

Data is observed in the LiveData format and is displayed by a recycler view.


Key Features


  • Updates the data by genre

  • Provides endless scrolling with paging

Movie Details

Displays the details of a selected movie and provides some additional functionality such as adding the to your favorites list.

Data is observed in the LiveData format and CRUD functionalities are handled through the view model.


Key Features


  • Displays the movie details

  • Shares the movie's IMDB page link

  • Directs the movie's IMDB page

  • Plays movie trailer on YouTube

  • Displays multiple lists of movies from various endpoints

  • Displays a list of similar movies

TV Shows List

Displays a glimpse of search results for trending tv shows for various streaming networks.

Data is observed in the LiveData format and is displayed by a recycler view.


Key Features


  • Displays top trending tv show as a showcase

  • Inserts showcase tv show into local Favorite database

  • Plays showcase tv show's trailer on YouTube

  • Displays multiple lists of tv shows from various endpoints and networks

TV Show Details

Displays details of a selected tv show such as release date, rating, and overview, along with useful links to IMDB page and tv show trailer on YouTube.

Data is observed in the LiveData format and is displayed by a recycler view.


Key Features


  • Displays tv show details

  • Inserts tv show into local Favorite database

  • Shares tv show's IMDB page link

  • Directs the movie's IMDB page

  • Plays tv show trailer on YouTube

  • Displays a list of similar tv-shows

Search

Takes a keyword input and displays a paged list of search results for movies and tv shows related to the keyword.

Data is observed in the LiveData format and is displayed by a recycler view.


Key Features


  • Displays a list of search results

  • Takes user input as a keyword

Favorites List & Details

Displays lists of favorite movies and tv shows in a ViewPager. Allows the user to change the binge status for list items, display details, and remove them from the lists.

The details screen displays a summary of details for the movie or the tv show.


Key Features


  • Displays a list of favorite movies and tv shows in a tab layout

  • Allows updating binge status for list items

  • Displays a limited edition of movie/tv show details stored in the local database

  • Allows displaying full details online
